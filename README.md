# 中国AI项目 🤖
张惠民的AI聊天网页项目

---

## 项目简介
这是一个基于HTML实现的简易AI对话网页，包含以下功能：
- 深色主题的聊天界面
- 消息气泡与打字动画效果
- 预设AI回复，支持模拟对话交互
- 适配手机和电脑屏幕的响应式设计

## 项目文件
- `index.html`：项目主文件，包含完整的前端代码和交互逻辑

## 如何使用
1.  直接用浏览器打开 `index.html` 文件
2.  在底部输入框中输入文字，点击「发送」或按回车键发送消息
3.  体验AI助手的回复效果

## 项目目标
记录我学习前端开发与AI交互的实践过程，后续计划接入真实的AI接口，让对话更智能。
